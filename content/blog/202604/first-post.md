# 🚀 开启全新的创作之旅

你好！这是本博客的第一篇文章。

这个博客采用了目前最前沿的技术栈体系：
- **框架**：Nuxt 3 (完美的全栈 Vue 体验)
- **内容**：Nuxt Content (直接用 Markdown 写博文，零基础生成静态/SSR站)
- **样式**：Tailwind CSS (通过 Typography 插件提供卓越的富文本排版)

## 🎉 强大的代码高亮

Nuxt Content 内置了 Shiki 进行代码高亮。你可以在 Markdown 中随心所欲地写代码：

```vue
<template>
  <div class="text-blue-500">
    <h1>Hello, World!</h1>
  </div>
</template>

<script setup>
const msg = ref('你好，世界！')
</script>
```

接下来，你可以将你的想法以 `.md` 文件的形式放在 `content/blog/` 文件夹下，它们就会自动被渲染出来！
